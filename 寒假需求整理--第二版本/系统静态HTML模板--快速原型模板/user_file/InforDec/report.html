3、样品检出情况

  3.1样品检出情况--除自报检

统计分析发现，2013年1月至12月，食品室检测涉及茶叶品种39种，主要有中国绿茶、红茶、晓起皇菊、信丰祥牌婺源绿茶、绿茶(取前五个)等。受检茶叶样品313批次，共检出农药样品60批次，样品检出率达20.12%；其中，鄣公山有机茶的样品检出数量最高，为8次

4、检出农药的种类与频次

 4.1检出农药的种类与频次---除自报检

经统计，共检测农药项次3320项，涉及农药残留物80种。其中，检出农药频次670频次，涉及检出农药种类11种，主要为联苯菊酯、啶虫脒、氯氰菊酯、三氟氯氰菊酯、硫丹，（前5个）等。滴滴涕检出频次最高，共158次。

6、风险评估

 6.1检测项目与限量标准对比情况

GB2763-2014限量标准中相关茶叶的农药项目有60项，食品室具备30个检测项目能力，占50%，还有30个检测项目无检测能力，占50%。

 6.2.检出情况与限量库标准比对

经统计，在313批次的茶叶样品中，有280批次样品没有超出GB2763-2014限量规定，占90%，有33批次样品检出超标，超标情况为中国绿茶中乐果、红茶中联苯菊酯。
